**Publisher's Description**  
  

GetWindowText is a very small and portable program with which you can read the
window text.

By clicking the left mouse button on the button with GetWindowText icon and by
holding down the left mouse button (drag), the read can be started.

Then simply hold your mouse over the window from which the text should be
read.

It can be read almost all the texts, e.g. from Edit, Static, Groupbox -
Controls, etc.

This program supports also directory trees (SysTreeView32) and list views
(SysListView32) to read the entire contents. When you release the mouse
button, the reading will be finished.
