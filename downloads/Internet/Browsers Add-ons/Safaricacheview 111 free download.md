**Publisher's Description**  
  

SafariCacheView is a simple utility for Windows that reads and parses the
cache file of Safari Web browser (cache.db) and displays the list of all
cached files in a simple table. Every cache information line includes the
following information: Filename, Content Type, URL, Content Length, Server
Name, Server Time, Expiration Time, Last Modified Time, Content Encoding, and
Referrer.

SafariCacheView also allows you to select one or more cache items and then
extract them into the desired folder or save the cache list into
html/text/xml/csv file.
