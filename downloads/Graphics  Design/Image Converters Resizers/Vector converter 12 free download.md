Vector Converter is a tool to transform SVG in VML, SVG in GIF, VML in GIF and
vice-versa, using XSLT language and PHP.

Vector Converter works in a browser window and is very easy to use once
installed. Simply select the file and the type of conversion needed and click
convert.

6 converters are implemented. XSLT technologies are used to convert SVG
documents into VML. Conversion from VML and SVG into GIF are performed by PHP
graphical libraries. Finally raster images can be translated or better
included into a vector image, in SVG or VML.
