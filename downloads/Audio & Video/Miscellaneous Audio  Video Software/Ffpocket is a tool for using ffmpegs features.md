FFpocket is a tool for using [ffmpegA's](http://www.ffmpeg.org/) features. It is mainly developed to support [JublerA's](http://www.jubler.org/) multimedia preview features. **Features:**

  * Audio decoding for a multimedia file (video/audio) 
  * Precise frame/time seeking in the multimedia files (including b-frames)
