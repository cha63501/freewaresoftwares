How to use:

The syntax is:

AVITester (FileName.Avi) (FOURCC) (ReportFile.Txt) [options]

All parameters enclosed by () are mandatory  
All parameters enclosed by [] are optional

Following options are available for use in Beta 1:

  * \- "-w" - Forces usage of Windows time counter instead of using Read TimeStamp Counter instructions. 
  * \- "-h" - Displays help screen. 
  * \- "-d" - Skip decompression test. Only the compression test will be done and no decompression times will be reported in the report file 
  * \- "-fx" - Do not perform CPU frequency measurement (available only if -w is not used) but force frequency to x (i.e. -f1700000000 sets frequency to 1.7Ghz)
